# Awesome Vibe Coding — Project Documentation

## Overview

This repository is a curated catalog of applications and tools built primarily through AI-assisted development (vibe coding). It follows the [awesome-list](https://github.com/sindresorhus/awesome) format — the entire product is a single `README.md` file with categorized links.

## Repository Structure

```
awesome-vibe-coding/
├── README.md                          # Main catalog — all curated entries
├── CONTRIBUTING.md                    # Submission criteria and guidelines
├── LICENSE                            # CC0 1.0 Universal
├── docs/
│   └── PROJECT.md                     # This file — project documentation
└── .github/
    ├── ISSUE_TEMPLATE/
    │   └── add-project.yml            # GitHub Issue form for project submissions
    └── workflows/
        └── lint.yml                   # awesome-lint CI check on push & PR
```

## How It Works

### Entry Format

Each project entry in `README.md` follows:

```
- [project-name](repo-url) - Short description. `technology` `ai-tool`
```

### Categories

Current categories (will expand organically):

| Category | Focus |
|---|---|
| SEO & Digital Marketing | Search engine optimization tools |
| Advertising & PPC | Ad management, bidding, campaign tools |
| Content & Copywriting | AI-assisted content creation |
| Analytics & Reporting | Dashboards, tracking, data visualization |
| Productivity | Task management, automation, workflows |
| Developer Tools | CLI tools, libraries, dev utilities |
| Data & Analytics | Data processing, ETL, analysis tools |
| Creative | Design, art, music generation tools |
| Education | Learning platforms, course tools |
| Games | Games built with AI assistance |
| Other | Everything else |

### Submission Flow

1. **Via Pull Request** — Fork, add entry to `README.md` alphabetically, submit PR.
2. **Via GitHub Issue** — Fill out the `Add Project` issue template form.

### Acceptance Criteria

- Public repository with a README
- Built primarily with AI/vibe coding
- Working project or has demo/screenshots
- No affiliate links or paid promotions
